Braille Number: 3

Composition of Braille 3

To represent the number '3' in braille, first, include the
**number sign (dots 3-4-5-6)**, then
follow it with **braille 'C' (dots 1-4)**.

Since it uses the same dot pattern as braille 'C',
it must be used with the number sign to be recognized as a number.

Reading Order

Braille '3' is formed by **dots 1 and 4**
of the 6-dot cell.

Start from the top left
and check the dots downwards.

Other Explanations

The number '3' represents three units.

It is an important braille number used in various contexts.

This braille character, based on a 6-dot cell,
has **dots 1 and 4** raised.
