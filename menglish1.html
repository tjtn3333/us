Braille Number: 1

Composition of Braille 1

To represent the number '1' in braille, first, include the
**number sign (dots 3-4-5-6)**, then
follow it with **braille 'A' (dot 1)**.

Since it uses the same dot pattern as braille 'A',
it must be used with the number sign to be recognized as a number.

Reading Order

Braille '1' is formed by **dot 1**
of the 6-dot cell.

Start from the top left
and check the dots downwards.

Other Explanations

The number '1' is the most basic number,
meaning a single unit.

It is one of the first numbers learned in braille education.

This braille character, based on a 6-dot cell,
has **dot 1** raised.
