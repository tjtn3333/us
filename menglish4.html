Braille Number: 4

Composition of Braille 4

To represent the number '4' in braille, first, include the
**number sign (dots 3-4-5-6)**, then
follow it with **braille 'D' (dots 1-4-5)**.

Since it uses the same dot pattern as braille 'D',
it must be used with the number sign to be recognized as a number.

Reading Order

Braille '4' is formed by **dots 1, 4, and 5**
of the 6-dot cell.

Start from the top left
and check the dots downwards.

Other Explanations

The number '4' represents four units.

It is a frequently used braille number in various fields.

This braille character, based on a 6-dot cell,
has **dots 1, 4, and 5** raised.
