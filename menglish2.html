Braille Number: 2

Composition of Braille 2

To represent the number '2' in braille, first, include the
**number sign (dots 3-4-5-6)**, then
follow it with **braille 'B' (dots 1-2)**.

Since it uses the same dot pattern as braille 'B',
it must be used with the number sign to be recognized as a number.

Reading Order

Braille '2' is formed by **dots 1 and 2**
of the 6-dot cell.

Start from the top left
and check the dots downwards.

Other Explanations

The number '2' represents two units.

It is a fundamental braille number used in various calculations.

This braille character, based on a 6-dot cell,
has **dots 1 and 2** raised.
